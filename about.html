THE PROJECT

Our team was tasked with building a fully functional LAMP stack Contact Manager. We developed the core layers of the application, including the remote server, the MySQL database, the PHP API, and the JavaScript and HTML front end. The final site features a complete user authentication system. Once a user registers and logs in, they get a personal dashboard where they can easily add, search, edit, and delete their contacts.
